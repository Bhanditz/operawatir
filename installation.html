Installing OperaWatir

Below are installation instructions for Windows and GNU/Linux and Mac.

Installing OperaWatir on Windows

  1. The first thing you'll need to do is to install Java.

  2. Second, you need JRuby. You can download this from jruby.org. Make sure you get JRuby 1.5.1 or above. Earlier versions will not work.

  3. With that in order, you can install OperaWatir. Open the Windows command prompt (“Start” → “Run” → cmd) and type the following: 

    jruby -S gem install operawatir

  4. If you are using Windows XP, you might need to install Windows Installer 3.1 Redistributable (v2) and Microsoft Visual C++ 2010 Redistributable Package (x86).

  5. And we are done! have a look at the tutorial for an introductory guide to OperaWatir.

    If you want to start Opera and connect automatically, please have a look at the Automatic Start page.

Installing OperaWatir on GNU/Linux or Mac OS X

  1. The first thing you need is Java. You can use this command to install it on a Debian-based operating system like Ubuntu: 

    apt-get install sun-java6-jre

    On Debian it's available in the non-free repository, but you can also use this deb package.

  2. Second, you need JRuby 1.5.1 or above. Earlier versions will not work. In Ubuntu, this is available from Natty Narwhal (11.04) with the following command: 

    sudo apt-get install jruby

    If you are using an older Debian-based operating system, you can use this deb package.

    On Mac, go to jruby.org and download JRuby manually.

  3. With that in order, you can install OperaWatir like this: 

    sudo jruby -S gem install operawatir

  4. And we are done! have a look at the tutorial for an introductory guide to OperaWatir.

    If you want to start Opera and connect automatically, please have a look at the Automatic Start page.
