fire_event(event, x = 0, y= 0)

Lets you fire an ECMAscript event on the current page. By supplying x and y arguments it will also allow you to fire off an event in a specified location on the page.

Arguments:

event:name of the ECMAscript/JavaScript event you would like to fire. (I.e. “onMouseOver”, “onMouseMove”, &c.)
x:(Optional.) x coordinate lets you specify where the event should take place on the page.
y:(Optional.) y coordinate lets you specify where the event should take place on the page.

locate()

Locate the element on the page. Elements can be located using one of the following selectors:

  • name
  • id
  • xpath
  • selector
  • text
  • href
  • index
  • value
  • class
  • tag_name

Raises: NoSuchElementException: if element is not found.

38:           @element = @container.driver.findElementByName(@what)
