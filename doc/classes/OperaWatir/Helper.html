load_environmental_variables()

Environmental variables (OPERA_PATH, OPERA_ARGS, OPERA_INSPECTR) will overwrite both the default configuration options for OperaHelper as well as the settings from the test suite-specific helper.rb file.

start_inspectr(pid)

inspectr is a tool that logs program crashes and freezes.

Note that the environmental variable OPERA_INSPECTR takes true (boolean), "true" (string), 1 (integer) and "1" (string) as parameters, while OperaWatir::Helper.inspectr only takes true (boolean).
