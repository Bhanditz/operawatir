Creates a new window in the browser. Note that Opera does not differentiate between windows and tabs. To directly create a Window object, pass in the Browser.

Examples:

- (Object, Boolean) screenshot(file_name, hashes = [], time_out = 2)

Saves a screenshot of the window to the filesystem of the server.

Parameters:

  • (String) file_name —

    The absolute path to the location where you want the screenshot saved.

  • (Hash) hashes (defaults to: []) —

    A list of hashes to compare. If the hashes match the hashes of what you are trying to take a screenshot of, this method will return a boolean value instead of an actual screenshot.

  • (Integer) time_out (defaults to: 2) —

    Specifies how long we should wait before timing out.

Returns:

  • (Object) —

    A ScreenShotReply object.

  • (Boolean) —

    True/false depending on whether you have given a hash as an argument.
