Watir 1 API reference

Regular methods

These methods all select the elements they are named after and, when provided with a single argument, search by ID.

browser.div # is the first <div> on the page
browser.div('content') # matches the element <div id="content">
browser.div(:id, 'content') # matches the element <div id="content">
browser.div(:class, 'fancy') # matches the element <div class="fancy">
browser.divs # matches all the  <div> elements on the page

Special methods

These methods have non-default search attributes, and match many element types, or specific inputs.

browser.button('Go') # matches <button value="Go"> <input type="submit" value="Go" /> and more…
browser.link('index.html') # matches <a href="index.html">…</a>
Method Default attribute Elements
browser.button, browser.buttons value <button> <input type="submit"> <input type="button"> <input type="image"> <input type="reset">
browser.checkbox, browser.checkboxes name <input type="checkbox">
browser.file_field, browser.file_fields name <input type="file">
browser.hidden, browser.hiddens name <input type="hidden">
browser.image, browser.images src <img>
browser.link, browser.links href <a>
browser.radio, browser.radios name <input type="radio">
browser.text_field, browser.text_fields id <input> <input type="text"> <input type="password"> <textarea>
