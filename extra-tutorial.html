Extra installation instructions

Install Opera

  • Visit the Opera Website
  • Click Download.
  • Click the Download button at the bottom of the page
  • Ubuntu Click Open with Ubuntu Software Center, then click install
  • Windows Click Open File...

Install Java on Ubuntu

  • Go to System > Administration > Synaptic Package Manager
  • Go to the menu Settings > Repositories
  • Click on the “Other Software” tab
  • Check the “Canonical Partners” box
  • Click the Close button
  • In the top left click the Reload button
  • Search for sun-java
  • Click the checkbox then Mark for Installation
  • Click Apply, and you're done!

Install JRuby ≥ 1.5.1 on Ubuntu

These instructions need git and curl installed and use the Ruby environment manager (rvm).

  • Open the terminal (Applications > Accessories > Terminal)
  • Enter the following commands: bash < <( curl http://rvm.beginrescueend.com/releases/rvm-install-head )
  • echo '[[ -s "$HOME/.rvm/scripts/rvm" ]] && . "$HOME/.rvm/scripts/rvm"' >> ~/.bashrc
  • source ~/.bashrc
  • rvm install jruby
  • rvm use jruby You will need to enter this line every time you want to use Ruby.
